# विचारमंच : मूल्यांकन प्रणाली

भारतरत्न डॉ. बाबासाहेब आंबेडकर विचारमंच, लातूर - स्पर्धा मूल्यांकन प्रणाली.
स्वातंत्र्य, समता, बंधुता, न्याय यांच्या विचारांवर आधारित वैचारिक, बौद्धिक, सामाजिक व सांस्कृतिक चळवळ.

## Setup
```bash
npm install
npm run dev
```

## Features
- Admin Management
- Competition Tracking
- Results Analysis
- Public Reports
